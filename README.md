# ReactJs CAR Application Front end 

## Prerequisite
* Nodejs v6+
* yarnpkg (optional)


## Getting Started
```sh
git clone  path

cd path

yarn install
    or
npm install

yarn start
    or
npm start
```

Open [http://localhost:3000](http://localhost:3000)<br>


## Major Libraries used
* ReactJs
* Redux
* create-react-app
* React Router 4
* Bootstrap 4
* redux-form
* React Boostrap Table
* Babel
* ESLint
* Test Runner: Jest
* Test Assertion: Jest
* toastr
* lodash
* Font Awesome: because Bootstrap 4 no longer suppies glyphicons
